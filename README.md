# FinCal - Financial Calculator App

FinCal is a comprehensive Android application designed to help users with a wide range of financial calculations. From mortgage and real estate to retirement planning and tax estimation, FinCal provides a suite of easy-to-use tools to manage your finances effectively.

## Features

The app is organized into several categories, each containing specialized calculators:

### 1. Mortgage and Real Estate
- Mortgage Payment Calculator
- Loan Amortization
- Refinance Calculator
- Rent vs. Buy Analysis
- And more...

### 2. Auto
- Auto Loan Calculator
- Cash Back or Low Interest Calculator
- Auto Lease Calculator

### 3. Investment
- Interest and Compound Interest Calculators
- Savings and Investment Calculators
- ROI (Return on Investment) and IRR (Internal Rate of Return)
- Bond and Mutual Fund Calculators
- Present/Future Value Calculators

### 4. Retirement
- Retirement Planning Calculator
- 401K and IRA/Roth IRA Calculators
- Social Security and Pension Estimators
- Annuity and RMD (Required Minimum Distribution) Calculators

### 5. Tax and Salary
- Income Tax Calculator
- Salary and Take-Home Pay Estimators
- Marriage and Estate Tax Calculators

### 6. Other Financial Tools
- Loan and Payment Calculators
- Currency and Inflation Calculators
- Sales Tax and VAT Calculators
- Credit Card and Debt Payoff Tools
- Student Loan and College Cost Calculators
- Business-related tools like Margin and Depreciation Calculators

## Tech Stack

- **Language:** Java / Kotlin
- **UI Framework:** Android XML Layouts / Jetpack Compose
- **Architecture:** Standard Android Activity-based navigation
- **Features:** Edge-to-Edge UI support, View Binding

## Getting Started

### Prerequisites
- Android Studio (latest version recommended)
- Android SDK 24 (Nougat) or higher

## Usage

1. Launch the app to see the main screen.
2. Tap "Get Started" to view the calculator categories.
3. Select a category to see the list of available calculators.
4. Choose a specific calculator to perform your financial analysis.
